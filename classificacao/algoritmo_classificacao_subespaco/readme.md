# Algoritmo de Classificação por Subespaços Explícitos

Esse é um algoritmo de classificação que visa implementar parte dos estudos de Algebra Linear com uma abordagem simplificada medindo a distancia do vetor em cada subespaço explícito treinado. 

## Algoritmo de Treinamento

- transformar os dados em vetores
- organizar os vetore em colunas
- aplicar o processo de ortogonalização da base usando Gram-Shimit

## Algoritmo de Classificação

- transformar o dado de teste em vetor
- calcular a distância de cada vetor
- a menor distância (menor erro) é classe pertencente à ele
